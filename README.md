South African Inflation Forecasting

This project builds a simple econometric model to forecast South African inflation using macroeconomic indicators.

Variables used:
- CPI inflation index
- Interest rate
- USD/ZAR exchange rate

Methods:
- Lagged variables
- Ordinary Least Squares regression
- Time-series forecasting

Tools:
Python, Pandas, Statsmodels, Matplotlib, Scikit-learn.

Output:
The model predicts inflation trends and compares them with actual inflation data.

Project Files:
notebook/ – Jupyter notebook with full analysis
data/ – raw economic datasets
report/ – full explanation of methodology
